# Frontend Pattern: Atomic Design

## Propósito
Metodologia de organização de componentes visuais inspirada na química, dividida em 5 estágios:

## Níveis
* **Atoms (Átomos)**: Elementos HTML básicos (botões, inputs).
* **Molecules (Moléculas)**: Grupos simples de átomos operando juntos.
* **Organisms (Organismos)**: Componentes complexos compostos de moléculas.
* **Templates (Modelos)**: Layouts estruturais de página.
* **Pages (Páginas)**: Instâncias de templates com dados de APIs.
