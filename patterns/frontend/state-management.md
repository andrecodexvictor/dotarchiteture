# Frontend Pattern: State Management (Gerenciamento de Estado)

## Propósito
Abordagens para gerenciar estados locais, globais, e compartilhados de forma previsível no frontend.

## Estrutura
* **Local State**: Estado isolado de um componente (ex: React `useState`).
* **Global State**: Estado compartilhado por muitos componentes não relacionados (ex: Redux, Zustand).
* **Server Cache State**: Estado que espelha os dados remotos de um servidor (ex: TanStack Query/React Query).
