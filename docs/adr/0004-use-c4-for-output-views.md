# ADR-0004: Usar C4 Model para Visualização Gráfica das Saídas Recomendadas

## Status
Approved

## Contexto e Problema
Recomendar estruturas apenas em texto puro reduz a capacidade de absorção visual da arquitetura proposta pelos times e assistentes de IA.

## Decisão Proposta
Adotamos diagramas baseados no C4 Model (usando sintaxe Mermaid.js) nos relatórios de saída gerados para descrever as interações de serviços recomendadas.

## Consequências
* **Positivas**: Facilidade de renderização em qualquer arquivo Markdown visualizável na IDE ou no GitHub.
