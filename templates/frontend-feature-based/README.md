# Template: Frontend Feature-based Layout

Estrutura recomendada para projetos frontend.

## Estrutura de Pastas
* `src/features/`: Componentes, hooks, e APIs isolados por escopo de negócio (ex: `features/auth/`, `features/checkout/`).
* `src/components/`: Componentes visuais globais (botões, modais genéricos).
* `src/hooks/`: Hooks React/Vue compartilhados globalmente.
* `src/utils/`: Funções auxiliares gerais.
